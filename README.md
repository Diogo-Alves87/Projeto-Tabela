## Como Executar o Projeto

    Antes de começar, você vai precisar ter o Python (versão 3.10 ou superior), Poetry instalados na sua máquina, Xampp instalado.

# 1. Clonar o Repositório

# 2. para rodar o código utilize:

    poetry install <- para instalar e começar a usar o poetry no codigo
    poetry run python banco_de_dados.py
# 3. Do banco de dados

    caso queira optar pelo uso do xampp(meu caso) instale o xampp e ao abrir, clique no botão 'start' no apache e no mysql.

    caso queira optar pelo uso de outro banco de dados, atualize as linhas 5~10 para as informações do seu banco dados

    (No momento ele ainda não verifica se a tabela já está criada, ficará disponivel em atualizações futuras <3)
